TinyUtils

by Daniele Olmisani.

very small but still useful command line tools (see LICENSE file)

basename

returns the last part of a given path (code)

cat

dumps the content of a file on the standard output (code)

cksum

computes a very simple checksum value on standard input (code)

hd

dumps the content of a file as hex digits on the standard output (code)

nl

dumps the content of a file on the standard output adding line numbers (code)

rot13

performs simple rotation 13 character translation (code)

yes

responds yes to any question (code)

zeros

continuosly outputs the value 0 on the standard output (code)

zeroc

counts 0 bits on the standard input (code)
